Quick Folder Move

Welcome to your updated Quick Folder Move, now fully compatible with Thunderbird 115. With this update comes a few adjustments that might reshape your workflow. To keep you in the loop, let's delve into these changes and keep any surprises at bay.

This version adopts a newer extensions technology (MailExtensions), which promises greater stability. This means you can carry on using this add-on version even with future Thunderbird updates without a hitch.

The flipside, though, is that this new technology doesn't support every customization that you're used to. But don't worry, the following sections will guide you on adapting your workflow to this new version. Enjoy your updated Quick Folder Move!

All in one place

In the previous version, you had a variety of ways to launch the Quick Folder Move popup—from the message context menu to keyboard shortcuts. The location of the popup would vary, depending on what elements of the UI were active at the time.

Now, your workflow is more streamlined. Use the button in the main toolbar no matter if you want to move, copy or go to a folder. This change makes your navigation more straightforward and less cluttered.

Moving in context

In addition to the main toolbar button, you also have the option to move messages directly from the message header. This way you'll be able to take action right in context, and you'll be ready to tackle the next message without delay.

Changing Muscle Memory

The new technology restricts available keyboard shortcuts to prevent unexpected command changes. This means we'll bid farewell to Shift+M, welcoming Ctrl+Shift+M for 'move' instead. Copy and Goto have undergone the same change.

But you're not without options. You can tweak these shortcuts in the add-on manager, and this version still allows for the old shortcuts. Revert back to the legacy shortcuts in this add-on's preferences, or use the shortcut manager to define your own.

If you have any questions, please reach out via the add-on's GitHub page. This is also where you can help improve future versions of the add-on.
